== [Unreleased]

=== Added
- Initial release of asdf-ghjk plugin
- Core scripts: `list-all`, `download`, `install`
- Platform support for Linux (x86_64, aarch64) and macOS (x86_64, arm64)
- SHA256 checksum verification for downloads
- GitHub API integration with rate limit handling
- Automatic retry logic for failed downloads
- Comprehensive error handling and logging
- Color-coded output for better UX
- Runtime dependency checking
- Support for GITHUB_API_TOKEN environment variable
- ShellCheck compliance for all shell scripts
- BATS test suite with unit and integration tests
- GitHub Actions CI/CD pipeline
- Comprehensive documentation (README, CONTRIBUTING, TROUBLESHOOTING, EXAMPLES)
- Optional scripts: `help-overview`, `list-bin-paths`

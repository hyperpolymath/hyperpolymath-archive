toc::[]

== Overview

=== What is Zig?

Zig is a general-purpose systems programming language and build system focused
on simplicity, safety, and performance. It provides explicit control over
memory, compile-time execution, and cross-compilation without hidden runtime
dependencies.

=== What is asdf?

{url-asdf}[asdf] is a universal version manager that allows you to manage
multiple runtime versions with a single CLI tool.

=== Why asdf-zig?

[cols="1,3"]
|===
|Feature |Benefit

|**Version Switching**
|Switch between Zig versions per project

|**Project Isolation**
|Pin Zig per repository via `.tool-versions`

|**Reproducible Builds**
|Align CI toolchains with local development

|**Cross-Platform**
|Works on Linux and macOS, with Windows via WSL
|===

=== Dependencies

[source,bash]
----
# Debian/Ubuntu
sudo apt-get update
sudo apt-get install -y curl git

# Fedora/RHEL
sudo dnf install -y curl git

# macOS (via Homebrew)
brew install curl git
----

=== asdf Installation

[source,bash]
----
# Clone asdf
git clone https://github.com/asdf-vm/asdf.git ~/.asdf --branch v0.14.0

# Add to your shell (bash)
echo '. "$HOME/.asdf/asdf.sh"' >> ~/.bashrc
echo '. "$HOME/.asdf/completions/asdf.bash"' >> ~/.bashrc

# Reload your shell
exec $SHELL
----

== Installation

=== Adding the Plugin

[source,bash]
----
asdf plugin add zig https://github.com/hyperpolymath/asdf-zig-plugin.git
asdf plugin list
----

=== Installing Zig Versions

[source,bash]
----
asdf list all zig
asdf install zig latest
asdf install zig 0.13.0
----

=== Setting the Version

[source,bash]
----
asdf global zig 0.13.0
asdf local zig 0.13.0
asdf current zig
zig version
----

== Usage

=== Project Configuration

[source]
----
zig 0.13.0
----

== Configuration

This plugin uses standard asdf behavior. No additional configuration is
required.

== Troubleshooting

* Ensure `zig` is on PATH after installation.
* Verify the installed version with `zig version`.
* Reinstall if a cached release is corrupted.

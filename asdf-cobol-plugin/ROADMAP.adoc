== Phase 1: Core Plugin Implementation

[cols="1,1,3"]
|===
| Status | Item | Notes

| ☐ | `bin/list-all` | Scrape GnuCOBOL release versions from GNU FTP/SourceForge
| ☐ | `bin/download` | Fetch source tarball with checksum verification
| ☐ | `bin/install` | Configure, compile, install GnuCOBOL
| ☐ | Dependency detection | Check for required build deps (gcc, libdb, libgmp, etc.)
| ☐ | Basic test suite | Verify installation produces working `cobc`
|===

=== Implementation Notes

GnuCOBOL build requirements:

[source,bash]
----
# Debian/Ubuntu
sudo apt install build-essential libgmp-dev libdb-dev libncurses-dev

# macOS (Homebrew)
brew install gmp berkeley-db ncurses
----

Version sources:

* https://ftp.gnu.org/gnu/gnucobol/
* https://sourceforge.net/projects/gnucobol/files/

== Usage

Each plugin directory under `plugins/` is a complete asdf plugin. Use with `asdf plugin add <name> <path>` or inflate to a standalone repo with `git-inflate`.

Example:

[source,bash]
----
# Add a plugin from the monorepo
asdf plugin add deno ./plugins/deno

# Or symlink for development
ln -s $(pwd)/plugins/deno ~/.asdf/plugins/deno
----

== Not Included

The following asdf-related repos are kept separate (meta-tools and indexes):

* `asdf-acceleration-middleware` -- meta-tool
* `asdf-control-tower` -- meta-tool
* `asdf-plugin-configurator` -- meta-tool
* `asdf-plugins` -- plugin inventory/index
* `asdf-ghjk` -- different project

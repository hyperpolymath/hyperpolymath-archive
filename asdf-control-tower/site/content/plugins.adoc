=== Plugin Standards

All hyperpolymath asdf plugins adhere to:

==== Security
* SHA256 checksum verification for all downloads
* Signed releases where upstream provides signatures
* No execution of remote scripts during install

==== Quality
* Comprehensive test suite using bats-core
* CI/CD pipeline with GitHub Actions
* Support for Linux (amd64, arm64) and macOS (Intel, Apple Silicon)

==== Maintenance
* Semantic versioning
* Automated dependency updates
* Responsive issue triage

=== Contributing a Plugin

Want to contribute a new plugin or improve an existing one?

. Fork the repository
. Follow the https://github.com/asdf-vm/asdf-plugin-template[asdf plugin template]
. Ensure all tests pass
. Submit a pull request

=== Plugin Request

Need a plugin we don't have? Open an issue on the https://github.com/hyperpolymath/asdf-control-tower/issues[control tower repository] with:

* Tool name and homepage
* Why you need asdf support
* Any special installation requirements
